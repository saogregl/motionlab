# Glossary

## Product

The durable engineering container for models, scenarios, runs, assets, and shared presets.

## Model

A versioned authored machine or mechanism definition containing bodies, datums, joints, and sensor assemblies.

## Scenario

A runtime recipe over a specific model revision, including environment, inputs, sensor activation, output policy, and solver settings.

## Simulation Run

An immutable execution artifact containing references, diagnostics, output catalog, and recorded artifacts.

## Datum

An authored local coordinate frame anchored to a body.

## Sensor Assembly

A first-class authored sensor entity mounted to a datum and configured for live, replay, and publication behavior.

## Channel

A typed runtime output stream or queryable output descriptor shared across live and replay modes.
